Welcome to CSC324 Web Programming!

This is an overview of the Web Programming course at Stonehill College. Visit Stonehill's homepage.

In this class, we'll learn about the entire web development stack, including front-end, back-end and database technologies.

Front-End:

Front End refers to the part of the web development stack that happens on the client computer. This involves the structure, the styling, and the behavior.

The languages used for front-end development are:

  • HTML
  • CSS
  • JavaScript

Back-End:

Back-End refers to the part of web development stack that happens on the server computer. This involves communicating with and updating the database, checking user privileges (authentication and authorization) and much more.

The languages one could use for back-end are myriad (e.g., Django in python, Ruby on Rails in ruby, PHP). In this class, we will use NodeJS to reduce the number of new languages we have to learn.

Databases:

Databases are where we store information that is likely to change (think passwords, emails, large databases of articles like on a news site).

Databases are also "hosted" on the server side. While they can be hosted on the same server as the back-end, it is generally good practice to host a database on its own server.

For this class, we will use MongoDB (a NoSQL database) to stick to that JavaScript lifestyle.
